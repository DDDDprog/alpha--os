Alpha OS Live

Welcome to Alpha OS, a Debian-based live system.

Default user: alpha
Password: alpha

Notes:
- NetworkManager is included for Wi-Fi/Ethernet.
- You have sudo access (passwordless) for the live session.
- Persistence is supported: boot with 'persistence' and a persistence partition labeled 'persistence' with persistence.conf (/ union).
- To install to disk, enable the optional Calamares profile (see README).
